### 🚀 Projetos Recentes & Experiência

#### 🛠️ Dashboard Builder (Low-Code/No-Code)
Arquitetura inovadora permitindo aos usuários montar painéis de dados dinâmicos via Drag and Drop.
- **Foco:** Configuração visual de conexões HTTP (com Autenticação OAuth/JWT/Basic), uso de variáveis globais e dezenas de widgets interagindo entre si (cross-filtering).
- **Stack:** React, TypeScript, Zustand, TailwindCSS, dnd-kit.

#### ⚙️ Core API de Dashboards Operacionais (C# / .NET 8.0)
Arquitetura backend construída do zero para atuar como o núcleo de inteligência (motor de BI) de dashboards de alta volumetria.
- **Foco:** Performance extrema na consulta de Stored Procedures usando Dapper, com processamento assíncrono avançado para evitar deadlocks.
- **Observabilidade:** Sistema robusto de telemetria estruturada nativa via Serilog, tratando logs no banco de dados como ativos de BI (com JSON) e disparando alertas de saúde e lentidão em tempo real via **SignalR**.
- **Mecanismos:** Middlewares inteligentes para auto-diagnóstico (`SystemHealthMiddleware`), ranqueamento de slow queries e serviços em background monitorando a saúde de _Linked Servers_ SQL.
- **Stack:** .NET 8.0, C#, Dapper, SQL Server, SignalR, JWT.

#### 📈 Aplicativo de Gestão Financeira Pessoal (Fullstack)
Desenvolvimento completo de um SaaS de gestão financeira, abrangendo desde o backend (API) até a interface do usuário.
- **Backend (C# / .NET):** API RESTful segura com autenticação JWT. Destaque para a implementação de criptografia forte (AES-256) no banco de dados para proteção de dados sensíveis (como cartões de crédito e anotações pessoais) e isolamento físico de arquivos por tenant (ContactId).
- **Frontend (React / Vite):** Interface moderna, responsiva e acessível utilizando Radix UI, TailwindCSS e Framer Motion para micro-interações fluidas.
- **Funcionalidades:** Controle de orçamentos, metas, dívidas, cartões de crédito e upload de comprovantes.

#### 📊 Dashboards de Produtividade e Gestão Operacional
Desenvolvimento de painéis administrativos focados em monitoramento de KPIs em tempo real para operações complexas.
- **Destaque:** Implementação de módulos como "Gestão de Pausas", "Produtividade de Agente" e "Visão Multi-Canais", lidando com renderização de alta performance e cálculo de volumetria em tempo real.
- **Qualidade:** Cobertura massiva de testes (+300 testes unitários e de integração usando MSW), garantindo estabilidade no processamento matemático e nas lógicas de dados.

#### 🤖 Automação, Bots e Health Checks
- **Chatbots:** Lógica de fluxo, fraseologia e integrações via API para sistemas de atendimento.
- **Saúde do Software:** Criação de bots automatizados (em PowerShell) para monitoramento diário do software, disparando a suíte de testes de forma agendada e emitindo relatórios de integridade.

#### 📝 Task Manager (Estilo ClickUp)
Projeto pessoal de um gerenciador de tarefas.
- **Desafio:** Modelagem de dados complexa para tarefas e sub-tarefas e interface intuitiva.

#### 📄 Relatórios Corporativos (SSRS)
- Criação de relatórios complexos com expressões condicionais, *Top N filtering* e agregações avançadas para suporte à decisão gerencial.
